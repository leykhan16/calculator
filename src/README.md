# 🦀 Rust CLI Calculator

A simple interactive command-line calculator built with Rust.  
This project demonstrates core Rust fundamentals including input handling, parsing, control flow, and pattern matching.

---

## 🚀 Features

- Accepts user input from the terminal
- Supports basic arithmetic operations:
  - Addition (+)
  - Subtraction (-)
  - Multiplication (*)
  - Division (/)
- Handles invalid operators
- Prevents division by zero
- Clean and readable structure for learning purposes

---

## 🧠 Concepts Covered

- Variables and data types (`f64`, `String`)
- Ownership basics (String handling)
- Standard input/output (`std::io`)
- Pattern matching with `match`
- Error handling using `expect`
- String parsing (`trim()`, `parse()`)
